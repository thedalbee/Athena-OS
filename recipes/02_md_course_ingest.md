# Markdown Course Ingest

Markdown 파일 여러 개를 수업 자료로 바꿀 때 쓰는 레시피입니다.

## 입력 준비

- 파일 이름을 순서대로 정리합니다.
- 각 파일이 한 단계인지, 여러 섹션인지 확인합니다.
- 선행 개념이 뒤 파일에 숨어 있으면 학습 순서를 재배열합니다.

## 추천 순서

1. 파일 제목만 보고 코스 개요 만들기
2. 각 파일의 핵심 용어 뽑기
3. 선행 개념 사전 만들기
4. 파일별 lesson 생성
5. 파일 묶음별 quiz 생성
6. 최종 cumulative review 생성

## 좋은 프롬프트의 조건

- "요약"보다 "가르쳐라"를 명시한다.
- "내가 이 내용을 전혀 본 적 없다"를 명시한다.
- "모르는 용어는 먼저 쉽게 설명하라"를 명시한다.
- "설명 후 확인 질문"을 강제한다.
