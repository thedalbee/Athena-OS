주어진 자료를 바탕으로 전체 학습 코스를 만들어라.

반드시 포함할 것:
- 문서 전체 흐름 한눈에 보기
- 추천 학습 순서
- 초보자가 먼저 알아야 할 선행 개념
- 각 단계의 학습 목표
- 각 단계 뒤 확인 질문
- 전체 학습 시간 추천
- 가장 헷갈릴 포인트
- 마지막 총정리와 복습 루프

절대 하지 말 것:
- 문서 내용을 이미 안다는 전제로 압축 요약하기
- 용어 정의 없이 전문 용어만 연속 사용하기
- 자료의 구조를 그대로 붙여넣기만 하기

출력 형식:
- `코스 개요`
- `왜 이 순서로 배우는가`
- `1단계`
- `2단계`
- `3단계`
- `중간 점검`
- `최종 점검`
- `암기 카드 후보`
